Data Aggregation

Every KA Lite installation generates it's own unique ID as a part of the installation process. If the device with KA Lite installed every reaches an Internet connection, it will be able to contact our secure, central data repository and sync it's usage data.

This is great for larger projects that have multiple devices using KA Lite at the same time, because rather than having to travel to each deployment site individually, you can trickle sync the data via low-bandwidth connections, and view it all in one place in a secure online portal.

User Management & Administration

Similar to the process above, but in reverse! With Online User Management & Administration, you can create your teacher and student accounts all in one place, and then when the deployed device reaches an Internet connection, that information will be downloaded on the device, allowing a project administrator to configure and maintain multiple installation from one workstation in the cloud.

Data Backups

When your data syncs, we store it on our secure servers. This is great, because in the field, things don't always go as planned. SD cards become corrupted, lions eat laptops, and unfortunately, devices might disappear overnight, and without online data backups, so does the hard earned data of your learners.
